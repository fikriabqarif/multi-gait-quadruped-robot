## Related Work

This project builds on concepts from both academic research in legged robotics and open-source hobby implementations of multi-servo quadruped platforms.

### Academic References

1. **Feng Q., et al.**  
   *Rolling Locomotion Control of a Biologically Inspired Quadruped Robot Based on Energy Compensation.*  
   This work proposes control strategies for rolling locomotion using energy compensation, providing inspiration for gait logic and dynamic transitions.

2. **G. Bledt, et al.**  
   *Design and Control of a Robust, Dynamic Quadruped Robot (MIT Cheetah 3).*  
   This paper discusses mechanical design and control strategies for a state-of-the-art quadruped robot, offering insight into gait planning and locomotion stability.

   *Towards Dynamic Trot Gait Locomotion: Design, Control, and Experiments with Cheetah-cub.*  
   This work presents gait planning and experimentation for dynamic quadruped motion, which is relevant to understanding how stable walking patterns are generated.

### Open-Source Reference

- **anoochit/arduino-quadruped-robot**  
  https://github.com/anoochit/arduino-quadruped-robot  
  This repository provides a practical Arduino-based quadruped robot implementation with 12 servo control. While not identical in architecture, it offers a useful example of structuring multi-servo control firmware and hardware integration for hobby robotics.

These references provide context for both the theoretical foundations and practical implementations that inform this project.
